>Service Catalog:

The Service Catalog provides a user-friendly interface where users can request services, items, and access.
It contains categories, catalog items, record producers, order guides, and content items.

1. Service Catalog

Laptop Request

Each catalog item has:

Variables (fields)

UI policies

Catalog Client Scripts

Workflows / Flow Designer

Approval rules

3. Categories

Catalog items are grouped inside categories to keep the catalog organized.

Examples:

IT Services

HR Services

Facilities

Hardware

Software
