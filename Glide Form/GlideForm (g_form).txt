g_form is a client-side API used in ServiceNow to interact with form fields.
It works only in:

‚úî Client Scripts
‚úî UI Policies
‚úî Catalog Client Scripts
‚úî Record Producer Scripts

üîπ Why Do We Use g_form?

To control fields on the form, such as:

Make fields mandatory

Set and get values

Disable / enable fields

Display messages

Restrict user actions

GlideForm (g_form) ‚Äì Complete Method List

1. Get & Set Values
‚úÖ getValue(fieldName)

‚úÖ setValue(fieldName, value)

‚úÖ clearValue(fieldName)

‚úÖ setMandatory(fieldName, true/false)

‚úÖ setReadOnly(fieldName, true/false)

3. Messages
‚úÖ addInfoMessage(message)

‚úÖ addErrorMessage(message)

‚úÖ removeOption(field, value)

‚úÖ clearOptions(field)
